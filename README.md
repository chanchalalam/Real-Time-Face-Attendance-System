# Real-Time-Face-Attendance-System
This Real-Time Face Attendance System utilizes Python, computer vision, and Firebase to automate attendance tracking. By capturing facial images and comparing them to a database, the system can accurately record attendance in real-time. 

## Features
1. Real-Time Video Capture: Uses OpenCV to capture live video.
2. Face Detection and Recognition: Utilizes the face_recognition library for accurate face detection and recognition.
3. Firebase Integration: Employs Firebase Admin SDK for real-time database management and storage.
4. User-Friendly Interface: Displays attendance information with an intuitive interface.
   
## Libraries and Technologies
1. Python: The programming language used for development.
2. OpenCV: For video capturing and image processing.
3. face_recognition: For face detection and recognition.
4. Firebase Admin SDK: For database and storage management.
